## 关于

本项目仓库作为轻雪MC服务器反馈和对外更新日志
更新日志可在Release中查看

## 如何反馈?

向本项目仓库提交`Issues`并提供以下说明

> 你也可拉取PR至本仓库提供更为完善的请求, 支持文件格式
> `txt` `markdown(md)` `html(以及其css/js支持)` `json` `yaml` `toml` `py` `doc(x)` `xls(x)`

> 在反馈PR时需要使用Gitmoji.dev的表情以同步Liteyuki仓库标准

```markdown
## 反馈具体时间

20XX-XX(月)-XX(日)
XX(时, 24制或使用P.M.&A.M.)

## 遇到的问题或建议

具体内容

## 客户端信息(可选)

可选内容

```
